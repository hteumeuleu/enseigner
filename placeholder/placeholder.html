Acte 2

Faire un placeholder avec l'attribut HTML5 "placeholder"


<input type="text" placeholder="Prénom de la marquise">

En JavaScript, on peut ajouter un événement au clic sur l'input et agir sur l'attribut value.


//Fix unsupported placeholders
function placeHolder(id)
{
var demo = document.getElementById(id);

    demo.className = "fix-hint";
    demo.value = demo.placeholder;

    demo.onfocus = function()
    {
        if (this.className == "fix-hint")
        { 
        this.value = ""; this.className = "fix-nohint"; 
        }
    };

    demo.onblur = function()
    {
        if (this.value === "")
        { 
        this.className = "fix-hint"; this.value = demo.placeholder; 
        }
    };
    return false;

Quelles solutions se prêtent le mieux à cette modification ?

En CSS, On peut modifier la taille par défaut de chaque input pour lui donner une taille minimale plus large.


input {
    min-width: 200px;
}

En JavaScript, on peut récupérer la taille des caractères avec value.length et muliplier par la taille d'un caractère pour simuler la largeur parfaite.


var input = document.querySelector('input');
input.style.width = (input.value.length * 16) + 'px';
